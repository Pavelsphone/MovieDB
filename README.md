Инструкции по настройке:

Зарегистрируйтесь на https://www.themoviedb.org/ и получите API ключ

Замените "YOUR_API_KEY_HERE" в MainActivity на ваш реальный API ключ

Соберите и запустите проект

Этот проект будет:

Подключаться к TheMovieDB API

Получать список фильмов с высоким рейтингом

Преобразовывать JSON в Java объекты

Выводить информацию о фильмах в Logcat
